画像ファイルパス -
/ 再生速度fps / 表示フレーム番号
マウスカーソルを画像に重ねて ホイール操作でフレーム移動 [ Shift x10 / Ctrl x100 / Shift+Ctrl x1000 ]
画像フォルダパス :
画像ファイル名 : +フレーム番号+
フレーム番号 : 開始～終了, 桁数
【使い方】
・画像フォルダパス : Explorerからコピーしてきてペースト。
・画像ファイル名 : ファイル名の数字の前後の文字を入力。
・フレーム番号 : ファイル名の数字の部分の開始番号と終了番号を入力。
・再生速度 : 秒間の表示フレーム数(停止して再生したときに反映)。
・表示フレーム番号 : 現在表示中のフレーム番号。入力+エンターで変更可能。
・桁数 : フレーム番号を "00100" のように桁数を固定する場合は、桁数を入力。固定しない場合は0。
・画像を再生成したらリロードしてください。
【制限】
・推奨 : Google Chrome (他での動作確認はしていません)
・映像表示部分は960x540に固定しています。
